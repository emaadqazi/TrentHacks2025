## Overview

ResuBlocks is a resume building application with drag-and-drop functionality, designed specifically for students. The app helps users create, optimize, and customize their resumes with AI-powered suggestions and alternative bullet points.

## Features

1. **Resume Upload & Critique** - Upload your resume and a job description to get personalized feedback
2. **Drag-and-Drop Blocks** - Build your resume using visual blocks that can be rearranged easily
3. **Alternative Bullet Points** - Get AI-generated alternatives for your resume bullet points with drag-and-place functionality
4. **Job Posting Scraping** - Automatically scrape and store job posting data for personalized recommendations
5. **Visual UI Effects** - Beautiful animations and transitions for a modern user experience

## Tech Stack

### Frontend
- **React 19** with **TypeScript**
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **@hello-pangea/dnd** - Drag and drop library (maintained fork of React Beautiful DnD)
- **Framer Motion** - Animation library
- **React Hot Toast** - Toast notifications
- **Axios** - HTTP client
- **Lucide React** - Icon library

### Backend
- **Node.js** with **Express**
- **TypeScript** - Type safety
- **Axios** - HTTP client for web scraping
- **Cheerio** - Server-side HTML parsing
- **CORS** - Cross-origin resource sharing
- **dotenv** - Environment variable management
